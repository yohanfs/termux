Termux
=================================================================================

.. contents:: **Daftar Isi**

Install Termux
---------------------------------------------------------------------------------

- install Termux app (download dari Google Play)

Install Softwares
---------------------------------------------------------------------------------

- vim

::

    pkg install vim

- git

::

    pkg install git

Storage
---------------------------------------------------------------------------------

Membuat link antara storage Termux dan internal storage.

::

    termux-setup-storage
